# Frostpunk非公式日本語化MODツール

## 概要
[Frostpunk](https://store.steampowered.com/app/323190/Frostpunk/)の非公式日本語化MODツールです。<br>
有志による日本語翻訳パッチを適応することができます。

なお、公式日本語実装が予定されています。
